# Laboration 4 - Backend-baserad Webbutveckling Mittuniversitetet 


## 🌟 Syfte

Denna webbplats använder Fetch API för att konsumera en egenskapad webbtjänst. 
På webbplatsen kan användare registrera konto och logga in via autentisering med JWT för att hantera sessioner och ge åtkomst till skyddade resurser. 
Projektet syftar även till att demonstrera kommunikation mellan frontend och backend samt hantering av användardata i en databas.

## 🔗 Webbtjänst

Denna webbplats använder en REST-API som finns här:

https://lab4-backend-6ioa.onrender.com/

##  Kom igång

1. Klona repositoryt

2. Gå in i projektmappen

3. Installera dependencies:
npm install

4. Starta utvecklingsserver:
npm run dev
